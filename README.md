# Maxwell-AI (v0.1) 性能与架构实验短报

> 状态：v0.1 基线版（实验性质）
> 本文档是对项目整体白皮书 (`paper_draft.md`) 的极简概括摘要。

## 1. 项目简介

本仓库 `maxwell-pinn-mvp` 是一个基于 PyTorch 构建的 **物理信息神经网络（PINN）微型实验工作区**。项目的核心主线是围绕麦克斯韦方程组（Maxwell's Equations），通过将物理现象（位移电流 $\partial \mathbf{D}/\partial t$）的特性抽象到网络拓扑中，设计出一种具备物理可解释性的稀疏算力结构，以期从算法层优化神经网络（特别是针对 PDE 求解的 PINN）的高频全量计算浪费。

## 2. 模型结构：位移门控组件

我们设计了核心构造块：`DisplacementFieldCell`。其在保留 SIREN（正弦激活函数）表征高频信号优势的同时，引入了基于特征动态变化的门控机制，其正向传递伪代码如下：

```python
h = sin(W_h * u + b_h)       # 提取空间/时间下的特征变化（拟配位移电流）
g = sigmoid(W_g * u + b_g)   # sigmoid 动态门控（阈值基于梯度变化）
u_next = g * h + (1 - g) * u # 带物理门控的残差穿透
```

该门控的 L1 正则化设计结合常数截距初始化（$b_g = -1$），促使网络通过 PDE 物理残差反馈，自动识别时空中**电磁场变化剧烈**的区域（即门控开通），而在**静止/平缓期**直接走网络残差通路（即门控关闭），从而形成“动态稀疏激活（DSA）”。

## 3. 实验结果摘要

基于本机环境（含 RTX 3050 Laptop GPU 与 Android 移动端设备）验证，主要结论如下：

**精准度与算法内生稀疏特征：**

* 1D Maxwell 方程：使用 `Displacement-Gated` PINN 时，由于算法能捕获有效的波阵面，相比 `Baseline MLP` 其 L2 相对误差下降约 37.5%，最终门控稳定在 **73.6% 的自组织稀疏抑制**。
* 2D Maxwell (TM 模式谐振腔)：随物理模次复杂度增加，性能反超更显著，相比基线 L2 误差大幅降低达 **49.4%**，网络门控呈现出 **71.2% 的稀疏抑制率**。

**桌面级 GPU 与端侧 Edge 部署硬件性能：**

* **RTX 3050 (AMP 适用性)**：借由纯粹的 Linear 核心矩阵操作，上述模型对混合精度具有满额适应能力。从 FP32 切换至 AMP (FP16) 时，耗时与峰值显卡内存下降均 **近50%**（*Batch=8，平均 7.1ms/step，极低内存开销*）。
* **移动端物理部署 (Android)**：打通了 TFLite 的导出生态后，由量化（INT8）重塑的等宽拓扑在 Android 实机 CPU（4线程）上，将推断延迟猛降 **59% (21.4ms → 8.8ms/frame)**。完全符合甚至超越通用的 MobileNet CNN 开销级别。

*(完整的基准测试细节和跑分见 `results/perf_baseline.md`)*
