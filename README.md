# MaxWell: Displacement-Gated PINN

Maxwell-AI 是一个基于 PyTorch 的物理信息神经网络（PINN）实验项目，用来高效求解麦克斯韦方程。它通过“位移门控”结构，让网络只在电磁场变化剧烈的区域激活，大幅减少无效计算，同时在桌面 GPU 和 Android 端都有实测加速。

## 1. 核心架构与原理
本项目在标准物理信息神经网络（PINN）的基础上，创新引入了**位移门控（Displacement Gating）**机制：  
借鉴麦克斯韦方程组中的位移电流项 \(\partial \mathbf{D}/\partial t\) 概念，模型设计了一个轻量级的辅助网络分支，用于评估局部时空域内物理场的“活跃程度”。  
该分支动态输出一个掩码（Mask），在预测场变化平缓或稳定的区域（门控值低于设定阈值），直接截断对应的主网络前向计算，从而形成“动态稀疏激活（Dynamic Sparse Activation, DSA）”。

## 2. 工程与实验总结 (v0.1)

### 1D Maxwell 诊断实验
- 修复基线损失函数和 \(H_y\) 解析解边界后，1D L2 误差已成功降至合理的 \(10^{-3}\) 量级。  
- 位移门控模型（Displacement）相较于基线（Baseline），在精度不妥协的前提下展示出显著的稀疏化潜力。

### 2D TM 腔验证实验
- 在更高维、更复杂的 2D 谐振腔模式下，位移门控展现出更显著的优势（精度提升约 49.4%）。  
- 门稀疏度约 71.2%，验证了“物理先验在处理局部高动态场变化时，能通过自发稀疏化引导模型更高效优化”的预期。

## 3. 后续研发路线
- **跨平台与生态整合**：研究将当前 PyTorch 架构平替至国产算力栈。  
- **算力实盘检验 (Wall-clock Speedup)**：在 C++ / CUDA Kernel 或芯片仿真层落实门控阈值断路，测试高稀疏带来的真实加速比。  
- **互动 Demo 实装**：计划在 `web-demo` 目录使用 Web 技术，将 1D/2D Maxwell 场数据转换为互动仿真演示。  
- **技术白皮书润色与输出**：以 `paper_draft.md` 为核心支撑素材，作为资金对接与行政审批的技术摘要与归档依据。
